# Portal de Checklists

Menu estatico para acessar os dois sistemas publicados no GitHub Pages:

- CheckList - Granel: https://nicolasprior.github.io/checklist-granel/
- CheckList - Fracionado: https://nicolasprior.github.io/checklist-fracionado/

## Como publicar no GitHub Pages

1. Crie um repositorio novo, por exemplo `checklist-menu`.
2. Envie os arquivos `index.html` e `styles.css` para a branch `main`.
3. No GitHub, acesse `Settings` > `Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch `main` e a pasta `/root`.
6. Salve.

Depois da publicacao, o endereco deve ficar parecido com:

```text
https://nicolasprior.github.io/checklist-menu/
```
